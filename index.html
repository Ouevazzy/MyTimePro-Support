FAQ

Comment restaurer mes données ?

Vos données sont automatiquement sauvegardées sur iCloud. Il suffit de vous connecter avec le même compte iCloud sur votre nouvel appareil pour retrouver toutes vos données.

L'application est-elle compatible avec tous les appareils iOS ?

MyTimePro est compatible avec iOS 17.0 et versions ultérieures sur iPhone et iPad.
